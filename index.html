simfy2spotify

Converts a simfy playlist to a spotify playlist

Download this project as a .zip file Download this project as a tar.gz file

simfy2spotify Bookmarklet

Create bookmarklet:

  • Start your browser
  • Add a new bookmark
  • Copy code of file bookmarklet.min.js in url field
  • Save

Usage at Simfy website:

  • Browse a Simfy playlist
  • Click bookmarklet
  • A layer with copied playlist appear
  • The rest is self explained.
